  1. 协议全能手：完美支持Shadowsocks、VMess、Trojan等主流协议，甚至能实现混合代理策略

三、配置艺术：从入门到精通

成功安装只是长征第一步，真正的精髓在于配置。我们将配置过程拆解为三个层级：

基础配置（小白必看）

  1. 导入订阅链接：点击Profiles→点击+号→选择URL导入
  2. 启用TUN模式：Settings→Override→勾选"Enable TUN"（这是iOS系统的关键设置）
  3. 选择节点：进入Proxies标签页，点击延迟测试后选择最优节点

FAQ

Nekobox 是否支持多客户端节点管理？
支持。Nekobox 与 Shadowsocks、Clash 等客户端独立运行，互不干扰。用户可通过策略组灵活管理节点，实现多客户端分流和网络优化。
Nekoray 支持节点订阅的自动更新吗？
支持。用户可通过订阅 URL 批量更新节点，程序会自动下载并替换旧节点。定期自动更新可保证节点列表始终有效，减少手动维护的工作量，提高使用效率。
WinXray 的多策略组如何优化流量分配？
多策略组允许将节点按用途、延迟或地理位置分类，并结合规则文件自动分流流量。这样不同应用如游戏、视频和网页流量都能走最优节点，提高整体访问速度和稳定性。
WinXray 的节点延迟过高怎么办？
可通过策略组切换低延迟节点，或调整传输协议和端口。检查服务器网络和防火墙设置，确保网络通畅。必要时可启用 MUX 功能减少短连接延迟，提高整体网络体验。
Nekoray 如何导入 V2Ray 节点？
打开 Nekoray 客户端后，点击“导入配置”，选择 V2Ray 或 Clash 类型的订阅链接。程序会自动解析节点列表，用户可一键连接或进行延迟测试。
V2Ray 的 Routing 设置可以实现什么？
V2Ray 的 Routing 系统支持基于域名、IP、端口或协议的智能分流。用户可以指定特定网站直连、部分服务走代理，还能创建黑白名单，实现灵活的网络访问控制。
Matsuri 日志太多如何调整日志级别？
在 Matsuri 或其内核配置中修改日志级别参数（如从 debug 降为 info 或 warning），同时限制日志轮转与最大文件大小。生产环境建议关闭 debug 日志以减少磁盘 I/O 与性能影响。
V2Ray 如何设置多入口实现不同访问控制？
通过定义多个 inbound，对不同端口绑定不同的 tag 并在路由中匹配流量。可针对内网用户、远程用户或特定 IP 段应用不同规则，实现灵活访问控制与流量隔离。
PassWall2 自动切换节点机制如何工作？
PassWall2 会根据节点延迟和丢包率测试结果，结合策略组自动选择最优节点处理流量。无需用户手动切换，保证网页、视频和游戏流量的速度和稳定性，优化网络体验。
Potatso Lite 是否可导入节点二维码？
支持。用户可扫描节点二维码导入节点信息，也可手动输入节点参数。虽然不支持订阅自动更新，但对于轻量用户来说足够满足日常节点管理和流量分流需求。
